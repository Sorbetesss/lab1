Telnyx Notifier
===============

Provides [Telnyx](https://telnyx.com) integration for Symfony Notifier.

DSN example
-----------

```
TELNYX_DSN=telnyx://API_KEY@default?from=FROM
```

where:
 - `API_KEY` is your telnyx api key
 - `FROM` is your telnyx phone number
