Sweego Notifier
===============

Provides [Sweego](https://www.sweego.io/) integration for Symfony Notifier.

DSN example
-----------

```
SWEEGO_DSN=sweego://API_KEY@default
```

where:
 - `API_KEY` is your Sweego API key
