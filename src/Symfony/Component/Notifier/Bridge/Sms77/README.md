sms77 Notifier
=================

Provides [sms77](https://www.sms77.io/) integration for Symfony Notifier.

DSN example
-----------

```
SMS//_DSN=sms77://APIKEY@default?from=FROM
```

where:
 - `APIKEY` is your sms77 API key
 - `FROM` is your sender (optional, default: SMS)
